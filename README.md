**Data-Analysis-using-Excel** is a comprehensive Excel-based data analysis projects that explore insights from a survey of **data professionals** across the globe. The datasets we will work on include several fields both technical and  Finance in the data industry.

This project demonstrates **real-world Excel skills** including:
- Data cleaning and transformation
- Pivot tables and dynamic summaries
- Statistical analysis (mean, median, mode)
- Conditional aggregation and filtering
- Cross-tabulation (e.g., career switch vs. salary satisfaction)
